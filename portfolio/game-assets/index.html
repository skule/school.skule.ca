Game Assets

As part of the Lock Bypass Village, a physical security education group, I was asked to create graphics for several interactive learning games.

 I created images of security devices and electrical tools for teaching physical security vulnerabilities for a hands on experience at several online security conferences, including DEF CON and Grayhat. These images were then used to create a variety of original interactive games, where conference attendees could try different security exploits on different equipment. These games are still live and playable, and may require a basic background knowledge of circuits.
